# Personal SUI Learning Roadmap

***

## Day 1

### Introduction to Blockchains and DLTs and the Sui Ecosystem (A1)
- Duration: 1 hour
- Topics: Overview of decentralized ledgers, smart contracts evolution from Bitcoin to Ethereum, Sui ecosystem description, including Walrus, Enoki, Deepbook, and Seal

### Introduction to Sui Blockchain (A3)
- Duration: 1 hour
- Topics: An introduction to Sui and Move architecture, Validators vs Fullnodes, Sui Infrastructure

### Interacting with Sui (A4)
- Duration: 1 hour
- Topics: Creating a Sui wallet using CLI, using the Testnet Faucet, sending SUI on Testnet/Devnet, publishing your first smart contract, exploring transaction responses (JSON format)

### Basic Move Concepts (B1)
- Duration: 2 hours
- Topics: Introduction to Move, packages, modules, the Move compiler and Move testing, object types with and without the Drop ability

***

## Day 2

### Primitive Types (B2)
- Duration: 2 hours
- Topics: Numeric types, strings, pure arguments, dynamic (object) fields

### Objects Abilities (B3)
- Duration: 3 hours
- Topics: Copy ability and events, key and store abilities, option types, parameter passing (by reference and by value)

### Move Patterns (C1)
- Duration: 2 hours
- Topics: Capability pattern, one-time witness & publisher, hot potato pattern

***

## Day 3

### Sui Typescript SDK Basics (D1)
- Duration: 1 hour
- Topics: Initializing the Sui client, making first calls through the client

### Sui Typescript SDK – Read Queries I (D2)
- Duration: 1 hour
- Topics: Reading objects from the chain, reading dynamic fields

### Sui Typescript SDK – Introduction to Programmable Transactions (D4)
- Duration: 2 hours
- Topics: Submitting a simple transfer transaction, reading balance changes, gas and gas profiling

### Sui Typescript SDK – Advanced Programmable Transactions (E1)
- Duration: 3 hours
- Topics: Minting a custom NFT, reading object changes, handling errors, integration tests with Jest

***

## Day 4

### Sui Typescript SDK – Dapp Wallet (E2)
- Duration: 2 hours
- Topics: Integrating the wallet in applications, connecting wallet, displaying owned objects, executing a mint transaction

### Package Upgrades (H1)
- Duration: 2 hours
- Topics: Upgrade preconditions, versioned shared objects, upgrade requirements

### Move Patterns (H2)
- Duration: 3 hours
- Topics: Capability with properties, display pattern, witness pattern

### Discussion
- Topics: Working with students’ code, examples, questions, open discussion

***

## Day 5

### End to End Example App (F1)
- Duration: 8 hours
- Topics: Building a complete decentralized application: A custom coin dApp with a faucet, from requirements to production. Features include writing smart contracts in Move, event management, integration with React UI, and using a central shared object registry for Hero listing and token balance.

***

## Days 6–7

### Hackathon!
- Topics: Present, build, and experiment with practical applications for Sui and Move based on learned concepts

***

**Total Duration:** 40 hours

This roadmap emphasizes mastering Sui blockchain fundamentals, Move programming, smart contract operations, dApp development, frontend integration, and community skills required for modern blockchain environments.[1][5][6][9]
